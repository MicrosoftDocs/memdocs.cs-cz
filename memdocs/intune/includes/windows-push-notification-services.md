## <a name="windows-push-notification-services"></a>Služba nabízených oznámení Windows
V zařízeních spravovaných pomocí Intune, která spravuje použití správy mobilních zařízení (MDM), se vyžaduje služba Windows Push Notification Services (WNS) pro akce zařízení a další okamžité aktivity. Další informace najdete v tématu [Konfigurace podnikových bran firewall a proxy serverů pro podporu provozu WNS](/windows/uwp/design/shell/tiles-and-notifications/firewall-allowlist-config) .
